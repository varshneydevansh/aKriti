# aKriti Transparent Animation Starter Pack

Created from the provided aKriti logo image as the source.

## Design law preserved
- Heart shape is the lowercase `a`.
- Visible text is only `Kriti`.
- No extra `a`.
- No new decorations, underline, flowers, stars, 3D, gradients, or logo redesign.
- Transparent background throughout.
- Short seamless 4-second loop at 12 fps.
- Subtle squigglevision / boiling-line / hand-redrawn marker wobble only.

## Animation notes
The pack uses the original raster logo shape, then applies tiny per-layer loopable offsets, rotation, scale breathing, edge thickness variation, and local periodic wobble. Timing offsets differ per heart/letter so the motion feels organic and non-synchronized.
